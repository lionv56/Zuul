# Zuul

Een simpele textadventure.

## Het spel spelen

**Stap 1: Installeer dotnet**

Installeer de laatste versie van [dotnet](https://dotnet.microsoft.com/en-us/download).
Of installeer de LTS versie (Long Term Support).

**Stap 2: Start het spel**

Je kan het spel op 2 manieren starten.

Optie 1: Run Zuul.bat

Dubbelklik op Run Zuul.bat.
Of typ dit commando:

```
Run Zuul.bat
```

Optie 2: dotnet run

Open deze folder in de terminal.
Typ dit commando:

```
dotnet run
```

Het spel start nu.
Veel plezier!
